# LineTracing Car

A simple line tracing car simulation built with HTML, CSS, and JavaScript.

## Features

- Interactive line tracing simulation
- Responsive design

## How to Run

Simply open `index.html` in your web browser to start the simulation.

## Development

This project was developed using Google's Gemini AI for code generation and assistance.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
